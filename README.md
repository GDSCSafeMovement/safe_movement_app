# Safe Movement App

안전한 이동을 위한 서비스

## Development

### Git Clone
```bash
> git clone "https://github.com/GDSCSafeMovement/safe_movement_app.git"

> cd safe_movement_app
```

### Flutter
```bash
> flutter clean

> flutter pub get
```

### Firebase
```bash
> firebase init
/*
 * Select emulator and firestore
 */

> flutterfire configure
```
